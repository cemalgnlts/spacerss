Feed key: ${page.key}

All feeds are updated hourly.
API

Use this endpoint to receive updates in JSON format.

		GET /api/feeds/${page.key}

Returns:

    [
     {
      title: "Title",
      description: "Description",
      image: "image.png",
      link: "/link"
     }
    ]

Use the format parameter to get the data in xml format.

      GET /api/feeds/${page.key}?format=xml

Returns:

			<rss version="2.0">
			 <channel>
        <item>
         <title>Title</title>
         <desc>Description</desc>
         <image>image.png</image>
         <link>/link</link>
        </item>
       </channel>
			</rss>
Widget

There is also a small api to embed updates on your page.

      <iframe src="/api/feeds/${page.key}?format=html"></iframe>

A few options for customization.

      ?format=html
      &bclr=000000
      &clr=ffffff
